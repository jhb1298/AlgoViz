# AlgoViz 

AlgoViz is an interactive, highly intuitive web platform designed to demystify complex data structures and algorithmic concepts through beautiful, real-time visual simulations. Built with a focus on precision and performance, it helps students and developers build a strong mental model of how algorithms manipulate memory and state step-by-step.

##  Key Features

- **Interactive Execution Controls:** Play, pause, step forward, step backward, and adjust execution speeds dynamically to inspect edge cases.
- **Visual Concepts Implemented:**
  - **Sorting Algorithms:** Visual representation of time complexities (e.g., Bubble, Merge, Quick, and custom space-optimized sorting).
  - **Recursion & Backtracking:** Real-time state-space tree generation (e.g., Tower of Hanoi, N-Queens, Sum of Subsets).
  - **Memory & Code Visualization:** Step-through trace execution mappings showing stack variables and memory pointers.
- **Dynamic Input Generation:** Test custom array inputs, random distributions, or specialized edge-case matrices.
- **Dark Mode & Modern UI:** A sleek, human-centric user interface optimized for long debugging and study sessions.

##  Tech Stack

- **Frontend Core:** React 18, TypeScript
- **Styling & UI:** Tailwind CSS, Framer Motion (for smooth fluid state transitions)
- **State Management:** React Context API / Zustand
- **Deployment:** Vercel
